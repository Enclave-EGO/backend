## EGO-project

#### 1. Clone project

```bash
$ git clone https://github.com/huynhducthanhtuan/EGO-project.git
```

#### 2. Change directory to folder EGO-project

```bash
$ cd EGO-project
```

#### 3. Install dependencies

```bash
$ npm install
```

#### 4. Run in developement mode

```bash
$ npm start
```

### API Documentation: http://localhost:4000/api-docs
