## EGO Project - Backend

#### 1. Clone project

```bash
$ git clone https://github.com/Enclave-EGO/backend.git
```

#### 2. Change directory to folder backend

```bash
$ cd backend
```

#### 3. Install dependencies

```bash
$ npm install
```

#### 4. Run in developement mode

```bash
$ npm run dev
```

#### 5. Run in production mode

```bash
$ npm run build
$ npm start
```
